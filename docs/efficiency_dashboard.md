# Efficiency Dashboard

Use Efficiency to compare planned versus actual labor hours. Variance highlights over or under performance by period.

## Key metrics

- Planned hours
- Actual hours
- Variance hours and percent

## Tips

- Filter by labor category to focus on a specific team.
- Use the period range to match the cadence used in the IFS export.
