# Quick start guide

Welcome to PM Copilot. This guide walks you through the first 30 minutes: sign in, create a program, add a contract, bring in delivery data, and review your dashboard.

![Program dashboard overview](/assets/docs/programs-overview.svg)

## 1) Sign in and set your workspace

1. Use your company email to sign in.
2. Confirm your profile details on the **Profile & account** page.
3. Review the sidebar navigation so you know where Programs, Contracts, Imports, and Docs live.

## 2) Create your first program

1. Select **Programs** from the sidebar.
2. Choose **New Program**.
3. Enter a clear program name, customer, and timeline.
4. Save to open the program overview dashboard.

## 3) Add a contract

1. From the program overview, select **Add contract**.
2. Enter the contract code, start/end dates, and planned quantities.
3. Save to unlock delivery units and milestones.

Need more detail? Jump to the [Contract management guide](/docs/contract-management).

## 4) Import delivery data

PM Copilot supports three import types:

- [Cost imports](/docs/import-costs)
- [Milestone imports](/docs/import-milestones)
- [Delivery unit imports](/docs/import-delivery-units)

Each guide includes file formats, required fields, and troubleshooting tips.

## 5) Review dashboards

Use the program dashboard to check cost, delivery, and margin progress. The [Dashboard metrics guide](/docs/dashboard-metrics) explains every KPI and alert.
